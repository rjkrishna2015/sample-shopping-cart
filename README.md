
# Sample Shopping Cart
## Tech Stack

**Client:** React, Redux, React Bootstrap

**Server:** Node, Express

**Database:** Mongo DB


## Features

- Product Listing
- Search with Title or SKU
- Sort Filter for pricing
- Pagination
- Add to Cart
- Cart counter with show/hide Cart Item Listing
- Cart Item Listing
- Remove Item from Cart
- Display total amount
